# SimpleHtmlTaskBoard

Simple HTML taskboard, no installation required. Has saves/loads

Works on any Operating system.

You are ready to roll within few moments - no installation, no setting up.

## Installation

The *only* thing you have to do is:
Put this taskboard.html in any folder you want, but also in that folder create a new folder called Boards  - that's where boards will be saved/loaded from.

OR

You can just copy this whole repo somewhere and run the HTML from the folder - repo already has Boards folder
